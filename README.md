# Welcome to DeviceInfo!

![DeviceInfo](./Resources/DeviceInfoHeader.jpg)

DeviceInfoは、デバイスの情報を表示できる、Xamarin.Forms ベースの、クロスプラットフォームアプリです。

端末のモデルの名前からチップの情報まで、様々な情報を入手することができます。

バージョン1.0で大幅にコードが改良され、オープンソースになりました。
